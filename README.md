# Installation

## Env

(*REQUIRED*)

- SECRET_KEY
- DEBUG


## Local

```bash
git clone https://github.com/Gexilable/IMG2PDF
python3 -m venv venv
pip install -r requirements.txt
python manage.py runserver
```
